This repository consists of Utility Classes to be used in SAP.

***********************************************************

ZCL_BUFFER

Utility class for memory buffer

***********************************************************

ZCL_ITAB

Utility class for internal table operations

***********************************************************

ZCL_BC_BDC

Utility class for BDC (Call Transaction and Batch Input)

***********************************************************

ZCL_UTILS

Miscellaneous utility class with conversions, etc.

***********************************************************

ZCL_FILE_READER

This reads any file (at the moment CSV, XLS and XLSX are implemented)
and imports the data into a table, converting from external to internal
