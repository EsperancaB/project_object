ZCL_UTILS

Miscellaneous utility class

Methods:
CONVERT_INT_TABLE_2_CSV - This will convert your internal table into a CSV file
CONVERT_STRING_2_SOLI - This will convert your string into a SOLI type table (for e-mails)
CONVERT_TLINE_2_SOLI - This will convert your TLINE table (SAP Script) into a SOLI type table (for e-mails)
CONVERT_TLINE_2_STRING - This will convert your TLINE table (SAP Script) into a string
